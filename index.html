SW 개발시간 단축

표준화되지 않은 방식을 요구사항/설계 작성 툴을 통해 표준화하여 개발 시간을 대폭 줄입니다.

SW 복잡도 해소

인력 의존 탈피

하나의 통합 툴로 체계적인 시스템을 구축하여 특정 개인에 의존하는 환경을 해소합니다.

Core Features

OEM 요구사항 접수부터 납품까지
전체 프로세스를 자동화

사양 작성부터 검증까지 4가지 핵심 도구가 전 과정을 지원합니다.

1

요구사항 작성 툴

OEM 요구사항을 자비스 양식에 맞게 입력하면, 나머지 요구사항을 자동으로 작성합니다.

2

설계 작성 툴

요구사항에 맞는 아키텍처를 설계하고 표준에 부합하는 설계 문서를 생성합니다.

3

기능/통합 자동생성 엔진

HW 제어 설정, 코드, 컴포넌트를 자동으로 생성하여 핸드코딩 없이 개발합니다.

4

단위 검증 자동생성 엔진

로직/HW 제어 컴포넌트 단위의 검증 코드를 자동 생성하여 품질을 보장합니다.
